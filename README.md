# 📊 Bank Loan Campaign - End-to-End Data Analytics Project

## 📝 Project Overview
This repository contains an end-to-end data analytics project designed to optimize a bank's personal loan marketing campaign. The objective is to identify the core customer profiles that are most likely to accept personal loan offers, thereby improving conversion rates and minimizing marketing costs.

## 🛠️ Tools & Technologies Used
* **Excel:** Initial data exploration, data cleaning (handling negative experience years, missing values).
* **Oracle SQL:** Deep-dive analysis, customer segmentation, and performance metric calculations.
* **Power BI:** Dynamic and interactive 2-page dashboard for management reporting.

## 📁 Repository Structure
* `/Data`: Contains the cleaned dataset (`loan dataset final.csv`).
* `/SQL_Scripts`: Contains production-ready Oracle SQL queries (`loan_dataset.sql`).
* `/PowerBI_Dashboard`: Contains the final Power BI report (`loan-dataset.pbix`).
